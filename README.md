# AI Placement Prediction System

An AI-based web application that predicts student placement probability using machine learning.

## Features
- User registration and login
- Role-based access (Admin / Student)
- Placement prediction using ML
- Feature importance visualization
- Probability trend analysis
- PostgreSQL cloud database
- Flask web application
- Deployed on Render

## Tech Stack
- Python
- Flask
- Scikit-learn
- PostgreSQL
- HTML, CSS, JavaScript
